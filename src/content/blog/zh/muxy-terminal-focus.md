# Muxy — 同时处理多个项目时拯救你注意力的终端

如果你经常同时处理 4–8 个项目，你一定深有体会。我以前的做法是把一堆窗口散布在桌面上，随时盯着——说实话，这种方法需要习惯。

然后我发现了 [Muxy](https://github.com/muxy-app/muxy)。这是一个每个有追求的 vibecoder 都应该立刻尝试的终端。这个工具确实提升了我多任务时的专注度——工作变得明显更舒适了。

一个小提醒：macOS 会对下载的构建版本发出安全警告。但我们是工程师——源代码完全开放。你可以让 LLM 做一次安全审计，然后自己用 Xcode 构建。

有趣的是，我之前尝试过用 vibecode 方式开发自己的类似工具——[Open Agent Manager](https://open-agent-manager.suenot.com/)。但实际上，用 Web 技术构建终端的效果差强人意。体验不如预期。这类工具需要原生实现（好在现在各语言都有了），我一开始就应该用原生方案。

最近看到 Warp 开源后，我开始逐步把我的作品迁移到 Warp 上。也许有一天我会完成基于 Warp 的解决方案。但目前——我强烈推荐试试 Muxy。它真的很有帮助。
